wact-button

General-purpose button with three variants, disabled state, tooltip, and a loading/confirm API: startLoading(), stopLoading(confirmText?, duration?), reset().

Variants

Default Primary Destructive

Disabled

Default Primary Destructive

Tooltip

Hover over each button to see the tooltip.

Loading & confirm

Click a button to start loading. After 2 seconds, stopLoading() is called. The third button uses duration: 0 to hold the confirm state indefinitely — click Reset to restore it manually.

Load → reset Load → confirm (1.5s) Load → confirm (5s) Load → hold confirm Reset

Custom padding

Override internal padding with the --btn-padding CSS custom property.

Full width

wact-button is inline-block by default. Set display: block on the host to fill its container.
